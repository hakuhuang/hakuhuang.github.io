【转载】电脑版微信缓存清理路径

教程工具 转载
2020-04-17

电脑版微信存储空间默认存储在C盘的：C:\Users\你的用户名\Documents\WeChat Files\你的微信号

  • attachment 文件夹里是图标图片
  • backupfiles 文件夹里是备份的聊天记录
  • files 文件夹里面都是你传输的word、excel、pdf 之类的文件
  • filestorage 文件里是你选择存储了的文件
  • cache 文件夹是缓存文件
  • file 文件夹里是word、excel、pdf、压缩文件等等
  • video文件夹里是视频和视频封面
