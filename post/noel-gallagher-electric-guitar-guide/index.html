【笔记】如何分辨诺有缸看上去一模一样的电吉他

· 2020-07-29 · # Oasis # 音乐 # Noel Gallagher

前言

本文是个人考据Noel电吉他型号的笔记。网络上一些以前的考据文章和翻译，有的配图和翻译均存在错误。我核实然后整理了一些比较主要的电吉他记录下来，也加入了一些自己添油加醋的内容。如果存在差错请告知。看完本文后，你应该会发现Noel看上去一样的吉他，其实都挺不一样……

Epiphone Les Paul

使用时期：早期-1994
型号：Standard
颜色：Cherry Sunburst（又译：番茄炒蛋色）

1992年绿洲在Boardwalk排练

摄于1994年1月的早期演出

1993年Noel在曼城的Out Of The Blue Studios创作《Shakermaker》

1993年Noel在Monnow Valley Studios录制首专

《Supersonic》的单曲封面左侧钢琴前的这把

《Supersonic》的MV中也用了这把琴

Epiphone Les Paul Standard是Noel在绿洲早期演出最常使用的一把琴，因为当时他的积蓄只能承担得起这把低端型号。这把琴也是随后录制了专辑《Definitely Maybe》的吉他之一。

Johnny Marr于2016年出版的自传《Set The Boy Free》中回忆了第一次见到Noel的场景——“Noel和大多数曼城的传奇能人志士一样，住在一个叫India House的黑色公寓里。我敲了门，当他让我进去时……我环顾了一下狭小的空间，Noel自豪地向我展示了他最值钱的两样东西：一把Epiphone吉他和一个大鱼缸。"

但随后，由于这把吉他现场演出时各种问题要调音很久，Johnny Marr便于1994年赠送了他一把自己的Gibson Les Paul（见下文）。原先的这把Epiphone Les Paul被Noel送给了一位熟识的朋友。

Gibson Les Paul

使用时期：1994
型号：1960 Special （在1952 Gibson Les Paul Standard Goldtop的基础上生产）
颜色：Sunburst

这把琴由Johnny Marr于1994年送给Noel。Noel用这把琴创作以及录制了《Live Forever》和《Slide Away》。《Live Forever》的MV中也可以看见这把琴的露面。

同年绿洲在Newcastle演出时，Noel用这把琴砸了一位跑到舞台上闹事的粉丝，于是造成琴颈的损坏。Johnny Marr听说后又送了一把别的琴……

这把和上面那把Epiphone Les Paul外形上的区别，在于琴头的标识以及琴头旋钮的材质（Epiphone低端款旋钮用的是仿玉石造型的白色旋钮，有点塑料质感，Gibson的便是金属材质的了）琴身的颜色差距和木材纹理因为摄影光线原因不太看得出来。况且这把的照片很少，不知道是不是Noel同年砸坏没修所以没用过。

Epiphone Sheraton

使用时期：1996年著名的Maine Road演出，以及其余零散的几次
型号：1993 Epiphone Nashville Series Sheraton
颜色：定制琴，重新上色为英国国旗图案

Maine Road第一夜

Maine Road第二夜

这把琴是Noel的前妻Meg Mattews送给他的礼物，在伦敦的吉他店Nightingale Guitars重新定制上色为英国国旗图案。虽然这把琴是大部分人听说NG的名字，脑中浮现的第一幅标志性画面，但其实本人只在很少的场合使用过。

大部分Sheraton琴的特色，是具有Frequensator™拉弦板（Frequensator Tailpiece）和迷你双线圈拾音器（Mini-Humbuckers）；琴头处镶有Sheraton系列独有的传统藤蔓“生命之树”的图案；指板处镶嵌的白色部分由珍珠贝和鲍鱼壳制作，有个倒三角形的缺口。

（注：Frequensator™拉弦板应用在Epiphone乐器上的历史最远可以追溯到上世纪30年代，这种拉弦板不仅给Sheraton吉他带来最长的延音，还带来准确的音准。迷你双线圈拾音器的声音比单线圈的更加丰满，比普通双线圈（普通Les Paul装载的拾音器）的声音细节更加丰富，带给乐手更广范围的音色可能性。）

Sheraton的外观特色

——注意！

市面上曾经发行过的同款Union Jack纪念琴有两款：

  • Epiphone Sheraton Union Jack

于2014年发行的限量复刻版Epiphone Sheraton。拉弦板、拾音器和琴头都是复刻上文中介绍的Noel的琴。

  • Epiphone Noel Gallagher Supernova
油管上一位死忠博主集齐了其中四把

于2005年发行的签名款琴其实并非同款Epiphone Sheraton，而是在Epiphone Riviera上改造的Epiphone Supernova系列，产自韩国。当年一共发行了五款颜色（黑檀色、日落色、樱桃红、曼城蓝、英国国旗）。

这一套琴的指板镶嵌的白色图案是分开的两个平行四边形。但是和上面的Sheraton相比，这套琴没有Frequensator™拉弦板，没有琴头的藤蔓雕饰，使用的也是普通的双线圈拾音器，所以不能算是当年完全的同款复刻。但是多了带NG签名的护板也好眼馋，还是具有一定收藏价值的。

Gibson ES-355

使用时期：90年代末-至今
型号：1960s Gibson ES-355
颜色：Cherry

2012年在Isle of Wight音乐节演出

这把琴一直是Noel的命中挚爱，从绿洲世纪初成员变动至今，将近二十年的时间里都有使用。一般拿来演奏一些非常绿洲大场面的那种体育馆摇滚歌曲。

本人曾经于2015年接受PremierGuitar采访时暴言称，“我在这把355上花的4000磅，是有史以来花的最值的4000磅。我对这把吉他实在是心生敬意。这把琴暂时位居我老婆之后，因为我能和老婆上床。如果我可以和这把琴上床，就真的不知道该怎么选了。”

ES-355是ES-335系列衍生的亚种。市面上这个家族的其他亚种型号有：335 / 339 / 345 / 355 / Trini Lopez / Dot等，想了解这个系列的具体差别请点击这里。

其中Noel用过的琴有345 / 355 / Trini Lopez。他拥有很多把345和355，下文会继续介绍。

（注：上文提到的Sheraton和Riviera，以及列侬用过的Casino，虽然看上去都是差不多爵士吉他，但其实是另外几个家族的。335的衍生、Sheraton和Riviera都是半空心琴，Casino是全空心琴。）

2009年解散演出后台砸坏的355

2009年绿洲解散的那天晚上发生的事情众说纷纭，有说是Liam砸的355，也有说是Noel自己抡了Liam。事实难以还原，但可以证实的是这把335被砸的真是堪称惨烈。

Gibson ES-345

使用时期：？-至今
型号：Gibson ES-345
颜色：Cherry

在采访中，Noel称平常带上路巡演的吉他有345和355。

Noel的355和345其实非常好区别。355是345的进阶豪华版，拥有看上去超级贵妇的Bigsby摇把、尖锐的菱形琴头图案以及从一品开始的指板镶嵌（当然并不是说你在网上看见的所有带Bigsby摇把的都是355，因为年份设计和个人改装都有所不同，这里只是就Noel的琴而言）

Noel的355和两把345

Gibson Trini Lopez

使用时期：本世纪初偶尔几次
型号：Gibson Trini Lopez
颜色：目测Cherry？

2002年一次杂志采访

2004年Glastonbury演出《Wonderwall》

Trini这把琴和上面几把相比是比较好辨认的。琴身上的孔是两个尖锐的菱形，指板上的镶嵌是三品开始的两个向外的三角形箭头，琴头是一个很Fender的倾斜形（其实是Gem最爱用的Firebird琴的琴头）。

Gibson Trini Lopez

虽然Noel比较少用Trini，只在几次采访和演出中零星出现过，据说这几次也是借Andy Bell的琴。但Andy和Gem经常用这把。

Nash Jazzmaster

使用时期：高飞鸟三专巡演至今
型号：Nash JM-63 Jazzmaster
颜色：见下图

三把Nash Jazzmaster

《Holy Mountain》MV

Noel多年来都钟情于Gibson和Epiphone，有时候也偶尔使用Fender的Stratocaster和Telecaster。但自从造月专开始，Noel在巡演中多次使用了Jazzmaster（特别是演奏一些迪斯科歌曲）但大多使用的不是Fender原厂，偏偏中意Nash的Jazzmaster。

Nash是近年来一个比较小众的牌子，以复古做旧款的吉他而出名，价格比起Fender高端型号更加实惠，但也经常被一些Fender发烧粉丝诟病为“就是卖一个复古的外观靠脸吃饭”。

造月专巡演出现过的Nash Jazzmaster有一把贴了笑脸贴纸的黑色琴、一把贴了星星贴纸的黑色琴以及一把淡黄色的琴。黑色的两把均为做旧款。一开始看到还以为是他不小心磕破的……

网上找到一些新琴的照片，做旧的像磕破了一样

事实上，Noel自录制二专的时候就开始使用Nash的吉他了，那时便购买了一些Nash的Stratocaster和Telecaster。自称是因为那几年自己一把非常昂贵的1963年产Stratocaster不幸被偷走，无法花相同的价钱再去买回弥补，于是买了较为实惠的Nash，使用感受却意外的好。

2019年接受guitar.com采访时称，“Fender和Nash的Jazzmaster我现在都弹。你懂吧，我从来都不是个Stratocaster之类的吉他的粉丝，但之前巡演的时候我看到了Nash的Jazzmaster便买了……现在我有将近20把Nash的吉他。”

Fender Jazzmaster

使用时期：高飞鸟三专巡演至今
型号：Fender Jazzmaster Professional
颜色：White

琴头上就可以看到Fender的标志。
